## Temas

### I. Node Shell

Para ejecutar código javascript en linea, use el siguiente comando para acceder al Node Shell

```bash
node
```

Se puede ejecutar código de un archivo .js usando el siguiente comando

```bash
node nombreDeMiArchivo.js
```
